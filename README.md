# 機械手臂控制器

這是一個使用 PyQt5 開發的機械手臂控制程式，可以透過 Serial 通訊控制 Arduino 機械手臂。

## 功能特色

- **圖形化介面**：直觀的 Qt5 介面，易於操作
- **即時控制**：四個馬達的角度可以即時調整（0-180度）
- **多種控制方式**：
  - 滑動條控制
  - 數值輸入框
  - 預設位置按鈕
- **Serial 通訊**：自動偵測可用 COM 埠
- **即時回饋**：顯示 Arduino 回應和通訊記錄
- **預設動作**：內建測試動作和歸零功能

## 安裝需求

```bash
pip install -r requirements.txt
```

## 使用方法

1. **啟動程式**：
   ```bash
   python robot_arm_controller.py
   ```

2. **連接 Arduino**：
   - 選擇正確的 COM 埠
   - 點擊「連接」按鈕

3. **控制馬達**：
   - 使用滑動條或數值框調整各馬達角度
   - 角度會自動發送到 Arduino
   - 也可以手動點擊「發送所有角度」

## 介面說明

### 左側控制面板
- **連接設定**：選擇 COM 埠並連接 Arduino
- **馬達控制**：四個馬達的個別控制
  - 馬達 0：底座
  - 馬達 1：大臂
  - 馬達 2：小臂  
  - 馬達 3：夾爪
- **控制選項**：批次控制和預設動作

### 右側狀態面板
- **連接狀態**：顯示 Arduino 連接狀態
- **通訊記錄**：顯示所有 Serial 通訊內容
- **當前角度**：顯示四個馬達的目前角度

## Arduino 通訊協定

程式會發送以下格式的命令到 Arduino：
```
angle1,angle2,angle3,angle4\n
```

例如：`90,45,120,0\n` 表示四個馬達分別轉到 90°、45°、120°、0°

## 注意事項

1. 確保 Arduino 已上傳對應的控制程式
2. 檢查 COM 埠是否正確
3. 首次連接時 Arduino 會初始化 2 秒
4. 馬達角度範圍為 0-180 度
5. 關閉程式前會自動斷開 Serial 連接
